# Aoath的四种授权模式 #

- 授权码模式（authorization code）
- 简化模式（implicit）(client为浏览器/前端应用)
- 密码模式（resource owner password credentials）(用户密码暴露给client端不安全)
- 客户端模式（client credentials）(主要用于api认证，跟用户无关)
